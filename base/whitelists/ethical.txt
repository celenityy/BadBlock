# what?
## Not all data collection is evil.
## Analytics/Telemetry CAN be collected in a way that is privacy-respecting, WITHOUT tracking or harming users.
## We SHOULD reward developers who do the right thing, as it will give them an incentive to do so.

# BEFORE YOU RAISE YOUR PITCHFORKS:

# This list will ALWAYS be optional, these domains will ALWAYS be blocked on BadBlock lists where relevant
# These domains will NEVER be included on the Combined Whitelist.
# Again, these domains will NOT be excluded from the BadBlock blocklists.
# This is ONLY meant for users who want to reward developers for doing the right thing.

# CRITERIA FOR INCLUSION:
# For a domain to be added here, it MUST meet this strict criteria, NO EXCEPTIONS:

# * Data collected MUST not be able to identify or de-anonymize users.
# * Data collected MUST not be shared with 3rd parties.
# * Data collected MUST not be used for tracking or any kind of profiling in any way, shape, or form.
# * Data collected MUST be adequately protected against data breaches or any other type of security incident.
# * There MUST be a clear, easy way for users to opt out of said data collection.
# * If it is a website, it MUST disable data collection if a user has enabled Do Not Track and/or Global Privacy Control.

# Violation of ANY of this criteria WILL result in the removal of offending domains.

# If you believe any entries here do NOT meet this criteria, please file an issue.

# If you know of any other domains we should include, please file an issue.

# Now, let's begin.
